Language: C++
Compiler: g++

Description: The program creates a weighted, directed, graph by reading in information
from a file named "Ginput.txt". The user can read create the graph by typing "I". The
user can print the graph information by typing "W". The user can find the shorted path
from a vertice s to vertice t by typing C s t flag. The flag is 0 to print the length
of the shortest path, and 1 to print the shortest path itself.
